# Unpkg

Debian and rpm packaging for [Uncloud](https://uncloud.run). This now uses my [temporary
fork](https://github.com/miekg/uncloudplus).

No premade packages, just the code to create your own.

If you want pre-made Debian packages [look here](https://github.com/dariogriffo/uncloud-debian/), and specifically [the
releases](https://github.com/dariogriffo/uncloud-debian/releases).

## Corrosion

This is now an internal docker image, so no install (of the .deb package) is required.
